# Goals

MVP:
- Simple web UI where user writes a goal and extra context.
- App generates a ready-to-paste prompt for AI tools.
- Support a small fixed set of tools (ChatGPT, Cursor, Copilot, Gemini).

v1.0 ideas:
- Custom tools with names and roles.
- Saved prompt templates.
- Command history stored in browser.
- Export/import configuration as JSON file.
